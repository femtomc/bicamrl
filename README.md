## Quick Start

```bash
# Install dependencies
bun install

# Run everything
bun bicky
```

That's it! The server starts on port 3456 and the GUI opens automatically.

### Available Commands

- `bun bicky` - Run server + GUI
- `bun server` - Just the server
- `bun gui` - Just the GUI
- `bun test` - Run tests
- `bun kill` - Kill the server
